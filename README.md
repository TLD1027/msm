# how to use

主体程序位于文件夹 `cc` 中，主程序为`c.cc`

`server`文件夹存放测试使用的服务端，正式的服务端还在计划中

编译参数,需要c++11或者更高

```shell
g++ c.cc config.cc getfile.cc shell.cc -pthread
```

配置文件设置[请看此文件](./配置文件设计.md)

# 使用的第三方库

- httplib(用于发送http请求) 已经包含文件无需安装
- rapidjson(用于解析json) 已经包含文件无需安装
- boost(用于TCP连接) 需提前安装
